Project's Title : LaunchWebsite

Project Description : Cross Browser Testing 

How to Install and Run the Project:

    Install JRE and JDK
    
    Install Eclipse
    
    Install Maven
    
    Update System and Local variable in system configuration
    
    Install TestNG using eclise marketplace
    
    Import the project using Maven project 
    
    Run the project either with command line   mvn test -Dtest=Testcase1
    
    Or TestRunner.java ->Run As-> TestNG Suite

  
How to Use the Project:

     Cross Browsing Testing using TestNg and Maven.
